# RPL-Manajemen_Toko_Online

## Deskripsi
Project ini merupakan aplikasi **Manajemen Toko Online** berbasis **Laravel**, 
dibuat untuk memenuhi kebutuhan pembelajaran mata kuliah **RPL**.

## Fitur
- Manajemen Produk
- Manajemen Kategori
- Manajemen Transaksi
- Manajemen User
- Login & Logout
- Dashboard Admin

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap / Tailwind (opsional)

## Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` ke `.env`
4. Atur koneksi database
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
